# OptiFinish IA Workspace

This directory is the source of truth for the finalized information architecture.

## Structure

- `home/` - homepage role, sections, routing logic
- `products/` - full product portfolio grouped by ownership and brand
- `services/` - post-sale, support, maintenance, and service offerings
- `facility/` - manufacturing and R&D credibility layer
- `our-work/` - proof, deployments, case studies, testimonials
- `resources/` - content, newsroom, troubleshooting, video, downloads
- `about/` - company, brand, partnerships, leadership
- `contact/` - conversion and lead capture entry point

## Working Rule

Each folder should hold the planning notes for that IA element, including:

- `overview.md` - role of the section in the website
- content outlines
- page depth notes
- CTA and lead flow notes
- dependencies or assets needed
