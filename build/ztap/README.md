# Z-TAP Build Workspace

This folder contains the in-project Z-TAP source and its immediate working context.

## Structure

- `source/` - actual Z-TAP Next.js source code

## Role In The Main Project

Z-TAP is not only one of the product pages.
It is also the main reference for:

- branding direction
- interaction design
- section depth standards
- motion and CTA patterns
- Next.js implementation baseline

## Working Rule

When evaluating what should be reused for the broader OptiFinish site:

- use `build/ztap/source/` for the real implementation reference
- use `references/ztap/` for notes, strategy references, and extracted reuse logic
- use `documentation/` for project-level reports that describe Z-TAP's role in the wider ecosystem
